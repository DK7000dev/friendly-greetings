# Tudo com Carne Moída

Site oficial do infoproduto [tudocomcarnemoida.online](https://tudocomcarnemoida.online)

## Estrutura

```
/
├── index.html        # Landing page (página de vendas)
├── app/
│   └── index.html    # App de receitas (área do membro)
└── img/              # Imagens do site
```

## Tecnologias

- HTML, CSS, JavaScript puro (single-file)
- 129 receitas com carne moída integradas no app
- UTMify tracking (UTMs + Pixel)
